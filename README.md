# 🌦️ Weather App

A modern and responsive Weather Application built using **HTML, CSS, and JavaScript**.  
It provides real-time weather information using a weather API with a clean and user-friendly interface.

---

## 🚀 Features

- 🌍 Search weather by city name  
- ⌨️ Enter key support for quick search  
- 🌤️ Real-time weather data using API  
- 🌡️ Weather icons (dynamic updates)  
- 🌙 Dark mode toggle  
- 🕒 Live date & time display  
- ⏳ Loading animation while fetching data  
- ❌ Error handling for invalid cities  
- 🌄 Dynamic background based on weather conditions  
- 🧹 Clean and optimized JavaScript (no duplicate functions)  
- 📱 Fully responsive design  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- OpenWeatherMap API
